# 09-auth

This is one of the projects I worked on during my studies. It's a note-taking
app. A registration process has been implemented, after which users can fully
use the app. You can browse notes by type or search by keyword. The ability to
add new ones and delete old ones has also been updated.

## How to use

1. Copy this repository
   ```
   git clone (SSH or HTTPS key)
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Start the server on dev mode
   ```
   npm run dev
   ```
4. Or complete the project construction

   ```
   npm run build

   npm run start
   ```
